The ATM Management System is a console-based application developed in C++ that simulates the core functionalities of an Automated Teller Machine (ATM). This project is designed for learning and demonstration purposes, focusing on basic banking operations, file handling, and control structures in C++.

It allows users to securely interact with an ATM system by performing common transactions such as balance inquiry, cash withdrawal, cash deposit, and account management.
